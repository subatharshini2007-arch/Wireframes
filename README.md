# Wireframes
A mobile app concept that helps students learn smarter through adaptive quizzes, a personalized AI tutor, and a visual learning roadmap. Designed in Figma covering wireframes, high-fidelity UI, and a clickable prototype across 7 screens (Splash, Login, Onboarding, Dashboard, Smart Quiz, AI Tutor, Roadmap)
# Task 1 — Wireframes & Low-Fidelity Design

## App: Clariva — AI-Powered Education App

## Overview
Wireframes created for a mobile education app called Clariva. 
All screens are designed in gray-scale only, using placeholder 
boxes and labels to define layout, navigation, and content 
structure before any color or styling is applied.

## Tool Used
Figma

## Screens Designed
| Screen | Purpose |
| Splash | App intro screen with logo and loading bar |
| Login | Email, password fields and Google sign-in |
| Onboarding | Subject selection grid for personalization |
| Dashboard | Stats, AI insight banner and subject cards |
| Smart Quiz | Question card with 4 answer options |
| AI Tutor | Chat interface for AI-powered help |
| Roadmap | Visual learning path with locked/unlocked topics |

## Wireframe Principles Applied
- Gray boxes only — no color, no styling
- Every element labeled clearly (button, input field, card)
- Layout follows mobile-first design (390 x 844px frames)
- Bottom navigation bar consistent across all screens
- Content hierarchy defined before visual design begins
